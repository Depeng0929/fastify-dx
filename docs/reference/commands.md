# Fastify DX CLI Specification

## Initial setup: `dx setup`

Usage:

Specify relative or absolute directory:

```bash
% dx setup <dir>
```

If no directory is specified, it is assumed to be the current directory.

```bash
% dx setup
```

## Development mode: `dx dev <dir>`

Usage:

Specify relative or absolute directory:

```bash
% dx dev <dir>
```

If no directory is specified, it is assumed to be the current directory.

```bash
% dx dev
```
